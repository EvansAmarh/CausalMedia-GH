**Justification (for paper Section 2.x):**

> The Gaussian Copula model achieved an overall SDMetrics quality score
> of 0.9948 (column shapes: 0.9973, column pair trends: 0.9923) compared
> to CTGAN's 0.8846, and was therefore selected as the final synthesiser.

## 2. Statistical Validation

All five contextual variables in this dataset are categorical, so a
Chi-square goodness-of-fit test (not a Kolmogorov-Smirnov test, which
applies to continuous variables) was used to compare synthetic category
proportions against the real seed proportions.

| Variable | Test | Statistic | p-value | p > 0.05 |
|---|---|---|---|---|
| location | Chi-square | 2.7517 | 0.2526 | Yes |
| tablet_access | Chi-square | 0.3325 | 0.8468 | Yes |
| bandwidth | Chi-square | 1.3465 | 0.51 | Yes |
| resource_level | Chi-square | 4.4058 | 0.1105 | Yes |
| teacher_qualification | Chi-square | 2.319 | 0.5089 | Yes |

**All variables pass (p > 0.05): Yes**

## 3. Sampling Strategy

50,000 synthetic rows were generated by the Gaussian Copula model,
trained on a 10,000-record seed dataset (random seed = 42, fixed for
reproducibility). Downstream sampling for use in Role 4 should draw from
this 50,000-row pool as needed.

## 4. Caveats

- The distributions used to build the seed dataset (location, tablet
  access rates, etc.) are currently placeholders pending citation from
  GES/NCTE sources — see `docs/methodology.docx` and
  `src/generate_seed.py` for the exact values requiring citation.
- This validation confirms internal consistency between the seed and
  synthetic data. It does **not** validate the seed data's distributions
  against real Ghanaian schools — that validation is the responsibility
  of Role 2b (see `docs/ethics.md`).
