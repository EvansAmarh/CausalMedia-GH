# Ethics Declaration — Role 2 (Synthetic Context Generation)

**Governance path: Path D**

This study operated under Path D ethics governance: a validated synthetic
supplement to a publicly available, anonymised system-log dataset. No real
Ghanaian student records were processed as part of this role. This design
is consistent with the Ghana Data Protection Act, 2012 (Act 843), which
exempts processing of fully anonymised and synthetic data from individual
consent requirements.

## Scope of synthetic data use

The synthetic contextual variables generated in this role (school location,
tablet access, bandwidth, school resource level, teacher qualification)
were sampled from probability distributions informed by published Ghanaian
educational statistics (GES / NCTE reports — see `methodology.docx` for
full citations). No individual Ghanaian school, teacher, or student is
represented, referenced, or identifiable in this dataset.

## Supplement, not substitute

The synthetic contextual layer is designed to **supplement** a real
Ghanaian field validation subset (Role 2b, forthcoming) and serves as the
**training-phase context layer only**. It is not presented as, and should
not be interpreted as, a representation of real Ghanaian schools. Any
claims in the paper about population characteristics are framed as
approximations pending validation against real administrative data.

## Data provenance

| Source | Type | Ethics status |
