# CausalMedia-GH

A causal machine learning study estimating the effect of multimedia video engagement on student learning outcomes using the EdNet KT1 dataset.
- Unit of analysis: 10,000 students (pre-registered sample)
- Method: CausalForestDML (continuous treatment)
- Seeds: 42, 123, 777
- Robustness: DoWhy refutation + E-value sensitivity
- Explainability: EconML SHAP (no surrogate model)
Pre-registered on OSF.
